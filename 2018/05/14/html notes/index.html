html notes

<body></body>里面的内容是会展示在网页上的

  • 用<!-- --> 注释

  • <hx></hx>(x:1~6)标题标签

  • <em></em> 斜体强调

  • <span></span> 单独设置强调样式：span{}

  • <q></q> 短文本引用 自动加引号

  • <blockquote> </blockquote>整段引用 整段缩进

  • <br/>相当于一个enter想起来最近在淘宝上看到的程序员专用enter回车2333 所以这一行….被换了….（空标签 只有开始标签）

  • <hr/>也是空标签 默认是灰色较粗的分割线

  • &nbsp； 空格 一个空格就打一个&nbsp；一定要有分号呀 这算是捆绑的

  • <address> </address> 默认斜体 （然而这个有啥用？？和<em>有区别吗？？）.

  • <code></code> 加入单行代码（这个很丑= =）

  • <pre> </pre> 加入多行代码 （保留了原格式 但…也emmm）

  • 无序列表 （类似word里的项目符号 默认一个小圆点）

    有序列表 默认1. 2.

  • <div></div>独立逻辑 就是能看出来这是一个独立的部分

  • <div id="***">就是给自己看看的感觉 给每个部分起一个名字所以是id嘛 就在开始的时候加id 就可以了

  • <table> </table>在中间加入表格样式（就长的像表格吧…因为并没有线= =）

    • <table> 也可以替换<table summary="abstract">给自己的表格加一个摘要 仅仅为了增强可读性
    • 在table里面的内容见下
    • <caption></caption>标题（用于描述整个表格的）
    • <tbody></tbody>优先显示加载好的部分
    • <tr></tr>每行的两边加
    • <td></td>每一个单元格两边加
    • <th></th>表格表头（就是第一行的内容吧…..看上去似乎加粗了）

  • 用css样式为表格加入边框<style type="text/css">
    table tr td,th{border:1px solid #000;}
    </style>

  • <a href="网址" title="鼠标滑过时显示的文本">超链接使用 </a> 不过这个是默认在原窗口上打开页面

  • 在新建浏览器窗口中打开 <a href ="网址" target="_blank">显示文字 </a>

    以上这两种有什么区别吗

  • 插入图片(支持GIF PNG JPEGemmmjpg呢？)

    <img src="图片地址" alt="指定图像的描述性文本，当图像不可见时（下载不成功时），可看到该属性指定的文本 当然也可以不写" title ="鼠标滑过时的提示文本">

  • 表单

    • <from method="传送方式e.g.post or get " action="浏览者输入的东西被传送到的地方e.g. save.php"> </from> 也是一对出现的

    • <label for="username"> 用户名：</label> <label> –输入框前面的提示性文字 <label>是一对出现的

      • 文本输入框

      <input type="text" name ="username" vaule="为文本输入默认值"/> 最后有一个反斜杠

      • name 是给自己看的

      • value 是默认值

      • type 是输入类型

      • 密码输入框

        <input type="password" name="pass"/>

    • 按钮

      `

    • 多行文本输入

      <textarea rows="行数" cols="列数">文本</textarea>

    • 单选框

      <input type="radio" value="提交数据到服务器的值" name="控件名" checked="checked(默认被选中)"/> name要一样，否则并不能有单选的作用

    • 复选框

      <input type="checkbox" value="提交数据到服务器的值" name="控件名" checked="checked(默认被选中)"/>

    • 下拉列表框

      用<select></select> 围住选项 表示单选下拉列表；如果想多选可以改为<select multiple="multiple">

      <option vaule="提交值" selected="selected"显示的值</option> 有selected的表示该选项默认被选中

    • 提交按钮

      <input tpye="submit" value ="提交（显示的文字）">

    • 重置按钮

      <input type="reset" value="重置">

    • <label for="id控件">name</label> label标签不会向用户呈现任何特殊效果，它的作用是为鼠标用户改进了可用性。如果你在 label 标签内点击文本，就会触发此控件。就是说，当用户单击选中该label标签时，浏览器就会自动将焦点转到和标签相关的表单控件上（就自动选中和该label标签相关连的表单控件上）。

    • CSS样式表

    • 用{}包围， 里面用分号隔开，最后一个也可以加分号

    • 用/* */注释
    • 内联式 ：写在HTML<p>里面
    • 嵌入式：写在<style></style> 里面—–> <style type="text/css">span{color:red;}</style>
    • 外链式：<link href="base.css" rel="stylesheet" type="text/css"/> .css 要以英文命名 ，其他都是固定写法；这一段写在<head>里面
    • 以上三种的优先级为内联式>嵌入式>外部式（就近原则；css在相同权值的条件下）
    • 选择器：css的样式声明结构选择器{样式；} 比如<body>
    • 标签选择器：html代码中的标签。e.g. <html>、<body>、<h1>、<p>、<img>
    • 类选择器：前面是加英文的原点.；类选择器名称{css；}先用合适的标签比如<span> 先修饰要标记的内容；用class命名类选择器的名字，最好英文；可以用很多次；
    • ID选择器：前面是加#；<span id=""></span> 只能用一次
    • 子选择器(>)：用于选择制定标签元素的第一代子元素；e.g..foot>li{ ;}
    • 包含选择器：与子选择器类似，用空格表明选择后辈元素，而且不限定只有第一代子元素。
    • 通用选择器：用*指定匹配html中的所有标签元素*{ ;}
    • 伪类选择符：a:hover{color:red} 鼠标滑过的颜色
    • 分组选择器：.first,#second span{color:green;} 第一段和第二段中的span里的内容为绿色
    • 特殊性：标签的权值为1，类选择符的权值为10，ID选择符的权值最高为100。
    • 重要性：p{color:red**!important**;}
    • 下划线{text-drcoration:line-through}
    • 缩进text-indent:2em
    • 字间距：中文间距：letter-spacing;英文间距：word-spacing
    • padding是边框里的距离；margin 是盒子外的距离
    • 绝对位置position:absolute;相对位置position:relative;固定位置：position:fixed
    • 参照物写position:relative，需要根据参照物相对定位的元素写position:absolute;
    • 1em = 18px
